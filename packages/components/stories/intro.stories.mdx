# OpenUI-Kit

OpenUI-Kit provides you with guidelines, components and templates to build new features for OpenCRVS and high-quality branded applications, faster and better.

The guide is a living document created to meet the needs of OpenCRVS front-end developers and designers. If there is a component or pattern you need or you have any other feedback, question or comment please contact us.
